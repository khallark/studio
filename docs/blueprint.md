# **App Name**: OrderFlow

## Core Features:

- Authentication Interface: Home page with signup invitation and login option.
- Dashboard Aggregation: Dashboard displaying the combined order information from the connected Shopify store
- Shopify Connection: Allow users to connect their Shopify store using the Shopify API.

## Style Guidelines:

- Primary color: Deep Indigo (#4B0082) to convey sophistication and reliability.
- Background color: Very light lavender (#F0F8FF). It is from the same family of hue as the primary color, but highly desaturated, and also pale for a light mode scheme.
- Accent color: Violet (#8F00FF) for interactive elements.
- Font pairing: 'Playfair' (serif) for headlines, and 'PT Sans' (sans-serif) for body text.
- Simple, outline-style icons for navigation and key actions.
- Clean and intuitive dashboard layout with clear visual hierarchy.
